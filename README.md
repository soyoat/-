 ## 三星
 ###  三星定价俱乐部事件
#### 
  三星创始人李秉哲在三星发展的初级阶段就采取了野路子，就是造便宜货、做次品，利用成本低廉的残次品牟取暴利。那么，质量差的东西要怎么盈利呢？答案是操纵定价，改变行业规则。李秉哲机智地和自己的对手一起成立了合谋定价俱乐部。李秉哲在新加坡、台湾和日本等地设立了各个高管和运营经理参与的Glass Meeting，大家一起打高尔夫一起谈生意，他们就这样乐乐呵呵商量好了价格，一起赚消费者的钱。这种传统延续多年，直到2013年，“合谋俱乐部”才被曝光，韩国三星、LG因为和我国台湾奇美、友达等液晶面板企业涉嫌合谋操纵液晶面板价格，被国家发展改革委制裁3.53亿元，其中三星缴了1.01亿元。然鹅，这点罚款和三星从中获得的利益比，根本就是九牛一毛

###  三星抄袭苹果事件
#### 
  苹果iphone经过数年打磨后，一上市就风靡全球，一下子打破了原有的手机市场格局。作为业界老大，三星坐不住了。他们决定：做出自己的iphone。三星凭借自己多年手机制造的经验，整个产品工程团队加班加点，最后拿出了132页的iPhone产品特性详尽调研报告。有了报告之后，三星点对点模仿图标大小、Home键、边框，等等等，终于做出了Galaxy系列手机。不光抄袭，更无赖的是，三星利用自己注册专利多的优势肆无忌惮地践踏法律。2011年4月5日，乔布斯忍无可忍，在加州发起诉讼，控告三星侵犯iPhone与iPad的诸多专利权。面对天真的对手，三星表示：三星马上仗着自己专利多，在韩国、日本、德国、美国反诉苹果，称苹果侵犯其诸多移动通讯科技专利权。于是在英国、法国、意大利、西班牙、澳大利亚、荷兰等地，两大公司开始大打法律战。
这边在打官司，那边三星也紧锣密鼓地销毁证据。韩国的反垄断组织，有一次突袭三星在水原市的设施。他们遭到了门卫的的阻拦，于是马上报警。折腾了半小时进入之后，他们这才发现，三星员工们原来在焚烧文件、更换电脑！他们要阻止的时候已经来不及了。为了销毁证据，有的三星员工已经把文件直接吞到了肚子里……直接吞到……肚子里…………由于三星销毁了部分与苹果专利纠纷证据，被加州法院惩罚，但是证据已经销毁。在审问时，三星的技术人员就拒不认账，装作不知道蒙混过关最终，由于证据不足，几个国家法庭甚至做出了偏袒三星的判决：在首尔，法庭曾判决苹果侵犯三星两项专利，而三星侵犯苹果一项专利； 在日本，法庭曾不受理苹果的侵权诉讼请求，还要它支付三星的庭审费； 在英国，法庭曾做出偏袒三星的判决，理由是三星的平板“没有iPad那么酷”
虽然美国法庭判决三星赔偿苹果10.5亿美元，但是三星就多个小专利不断起诉，打算耗死苹果。那时的三星家大业大，消耗战完全不虚。但是苹果不行，无休止的专利战让苹果身心俱疲，最终苹果无奈放弃上诉，和流氓三星握手言和。但是苹果放过了三星，三星可不会放过苹果！此后苹果每次出一个机型，三星都会在自己的手机广告中嘲讽，已经成为了一个传统。三星嘲讽苹果的天线问题，嘲讽iphone4S用户排队买手机，甚至嘲笑iphone已经过时，是“老年人用的手机”。这种比城墙还厚的脸皮，也是没谁了！！！
